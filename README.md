## ðŸ“Œ How It Works

1. User enters height in **cm** and weight in **kg**
2. App converts height to meters and applies formula:  
   `BMI = weight / (height * height)`
3. Shows BMI value and category:
   - `< 18.5`: Underweight
   - `18.5 - 24.9`: Normal weight
   - `25 - 29.9`: Overweight
   - `30+`: Obese
